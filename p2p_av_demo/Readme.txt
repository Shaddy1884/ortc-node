In order to set up your own ORTC server (P2P call): 

- Install Node.JS (http://nodejs.org/)
- Run "npm install"
- Run "npm start"
- Allow access if prompted by firewall.
- Start the demo from IE10 through http://localhost:8888

	OR

- Install Node.JS (http://nodejs.org/)
- Install the Node modules listed below
- Run "node lib\gateway-p2p.js" to start the server.
- Allow access if prompted by firewall.
- Start the demo from IE10 through http://localhost:8888


Here are the nodejs modules that are required in order to run the demos and
tests within the SDK.  These modules must be downloaded and expanded into the
thirdparty directory that is at the same level as the "js", "build", and "demo" 
directories.

Each module lists the version, the directory it is expected to be expanded into
and the URL to retrieve the source for the module.
